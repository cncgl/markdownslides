# markdownslides

- make markdown to slides

## Requirement

- [Pandoc](http://pandoc.org/) (needs to be installed)

## Install

```
$ npm i
```

## Build

Write markdown files in `src` directory , image files put in `img` directory and execute:
 
```
$ npm run build
```
